exception Invalid_index

Invalid_index is an exception representing invalid index exception if the player input incorrect index when chow. Namely, the game raise invalid index exception if the player input incorrect index when chow

val chow_index_valid : t -> int -> int -> tile -> bool

chow_index_valid hand t1 t2 t3 is true if t1 t2 t3 is a chow. \n AF: the (hand : tile list) must already contain t1 and t2, otherwise it is not a valid chow. In other words, if a person chow 1 2 to 3 without having 1 2, system will retrun 'invalid command, no such 1 and 2'. else, here to determine if 1 2 3 forms a valid chow, and if not return 'invalid chow, 1 2 3 is not a chow' exception raise Invalid_index if the index is not valid

val pung_valid : t -> tile -> bool

pung_valid hand tile is true when tile can form a pung combination with hand and false otherwise

val kong_valid : t -> tile -> bool

kong_valid hand tile is true when tile can form a kong combination with hand and false otherwise

val ankong_index_valid : t -> int -> bool

kong_valid hand tile is true when tile can form a kong combination with hand using the tile at index pos and false otherwise

val winning_valid : t -> t -> tile option -> bool

winning_hand hand open_hand is true when hand is a valid winning hand in mahjong and false otherwise

val scoring : t -> t -> tile option -> int

scoring hand open_hand is the amount of points that is awarded to the player with the winning hand

val selfkong_valid : t -> t -> bool

selfkong_valid open_hand hand is true if one can use one of the hand to kong some three in the open hand. AF: the hand is a valid hand. if hand does contain such combo then return true

val selfkong_tile : t -> t -> tile

selfkong_tile hand1 hand2 is the tile in hand2 that can be self kong onto hand1

val ankong_valid_new : t -> bool

ankong_valid_new hand is true if the hand has a valid hidden kong. automatically kong the first hidden kong find. AF: the hand is a valid hand. if hand does contain such pattern then return true

val ankong_tile_opt : t -> tile option
