Module GameState

module GameState: sig .. end

Representation of the dynamic game at a particular instance

This module is the state of the game. It keeps track of the current house out of the four players, the points for each player, distance from termination, and the number of rounds that has elapsed.

type t = {
   round_num : int;
   termination_distance : int;
   scores : int array;
   players : Players.player list;
   house : Players.player;
   house_streak : int;
   house_index : int;
}

t is the state of the entire games

type game_progress =
| Quit of t
| Continue of t

game_progress is a wrapper of the game states that indicates whether the game should continue or quit

val index_of_player : 'a list -> 'a -> int

index_of_player player_list player is the index of the player in a list of four players players

val random_house_index : int

random_house_index is a random interger from 0 to 3

val players : bool -> Players.t

players is_advanced is the advanced npc if is_advanced is true, else basic npc

val init_game : int -> bool -> t

init_game distance is_advanced starts a new game state that represent the entirety of the mahjong game. distance is the amount of rounds that will be passed before the game terminates. is_advanced decides whether the user will play with three advanced level npc or three basic level npc

val calculate_score : t -> 'a -> 'a -> int

calculate_score t house winner is the score that will be added to the winner and subtracted from the loser(s) of the last round

val update_score : t ->
       Players.player -> Players.player -> Players.player option -> int -> unit

update_score t house winner losers tile_score is the game state t with updated player scores

val reached_termination : t -> bool -> bool

reached_termination t house_wins is whether the current game state is at termination or not

val continue_or_quit : t -> 'a -> bool -> game_progress

continue_or_quit t house house_wins returns the updated wrapped game state in Quit or Continue

val update_game_state : t -> RoundState.round_end_message -> game_progress

update_game_state t winning_message is the updated game state after a round has ended. It is wrapped by game_progress to indicate the updated state should quit or continue next

val update : t -> game_progress

update_round_results t results takes in the current game state t and the results of the last round results and return whether the game will continue and initializa a new round or end the game

val get_score : t -> int array

get_score t is the scores of the four player in the current game state t

val get_round : t -> int

get_round t is current round number of the current game state t

val string_of_scores : t -> string

get_score t is the pretty formatted string of scores of the four player in the current game state t
